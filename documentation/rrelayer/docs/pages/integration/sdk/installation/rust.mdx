# Rust Installations

Installing rrelayer is super quick and easy

:::code-group

```bash [github]
cargo add --git https://github.com/joshstevens19/rrelayer --package rrelayer
```

```bash [crates.io]
cargo add rrelayer
```

:::

you can now use the SDK, to get started with that go [here](/integration/sdk/create-client-authentication/rust).
