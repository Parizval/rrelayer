# Dashboard

The repo for rrelayerr is a monorepo to make it easier to maintain all so it has all the other elements in there, the dashboard UI is location [here](https://github.com/joshstevens19/rrelayerr/tree/master/rrelayerr_dashboard_ui).

With the dashboard its your portal to be able to do everything within the UI itself. You can add new users to be allowed to login to the dashboard, currently we support login with github.

You can new relayers, config existing relayers, add new API keys, view transaction historys and more.
